# HP Poly Lens - Оновлена інформація

## Про проект:
- **Назва:** HP Poly Lens (раніше Plantronics HUB)
- **Компанія:** Hewlett-Packard / Poly
- **Платформи:** Android & iOS
- **Ваша роль:** Android розробник (основна), iOS підтримка протягом місяця
- **Play Store:** https://play.google.com/store/apps/details?id=com.plantronics.headsetservice
- **Офіційний сайт:** https://www.hp.com/us-en/poly/software-and-services/software/poly-lens/app.html

## Ключові функції (з скріншотів):
1. **Device Management**
   - Poly Voyager 60 підтримка
   - Статус підключення (Connected/Disconnected)
   - Моніторинг батареї (Left/Right/Case)

2. **Find My Device**
   - Локація на карті
   - Звуковий сигнал для пошуку

3. **Quick Guide**
   - Інтерактивний гайд по пристрою
   - Tap targets для навчання

4. **Settings**
   - Language
   - General
   - Software Updates (показує доступні оновлення)
   - Network
   - Ringtones & Volume
   - Sensors & Presence
   - Wireless
   - Advanced

5. **OTA Updates**
   - Автоматична перевірка оновлень
   - Версії прошивки (1.0.20 → 1.0.21)

## Технології:
- **Android:** Kotlin, Java
- **iOS:** Swift (підтримка)
- **Bluetooth:** BLE для зв'язку з пристроями
- **UI:** Material Design 3
- **Backend:** REST API для оновлень

## Ваш внесок:
- Розробка Bluetooth комунікаційного шару
- Імплементація UI для управління пристроями
- Робота над Find My Device функціоналом
- Підтримка iOS версії для досягнення feature parity
- OTA оновлення прошивки
