## 🌟 Overview

This portfolio showcases the Plantronics HUB Android application, featuring:
- Real-time headset monitoring and control
- Battery status tracking
- Wear detection (DON/DOFF)
- Android Wear companion app
- Support for 30+ Plantronics devices
- Modern Material Design interface

## 🚀 Quick Start

### Local Development
1. Clone or download the files
2. Open `index.html` in a browser
3. No build process required!

### Customization
1. **Colors**: Edit CSS variables in `:root`
2. **Content**: Update text in `index.html`
3. **Images**: Replace placeholder graphics
4. **Links**: Update download/GitHub URLs
