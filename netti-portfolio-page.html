Overview

Netti is a powerful and intuitive mobile application for Android, designed for network engineers, system administrators, developers, and enthusiasts who need advanced tools for network diagnostics, testing, and analysis directly from their smartphone or tablet. The goal was to provide Android users with access to professional network utilities typically available only on desktop computers, in a convenient and modern mobile interface.

Key Features

iPerf3 Bandwidth Testing

  • Client and Server modes implementation
  • Flexible test parameter configuration
  • Real-time result visualization
  • Saved server profiles with availability status

Ping

  • Latency measurement to specified hosts
  • Configurable test duration (including infinite mode)
  • Real-time ping change graph
  • Min/avg/max latency display

Trace Route

  • Network packet path visualization
  • Interactive map with node geolocation
  • Detailed hop information list
  • Comprehensive output log

Network Information

  • Detailed current network connection data
  • Internal/external IP address, subnet mask, gateway
  • DNS server information
  • Available network interfaces list

Device Data Transfer Rate Test

  • Internal data processing speed evaluation
  • Device information display

Test History

  • Automatic saving of all test results
  • Date and test type grouping
  • Detailed previous test results

Development Challenges & Solutions

Challenge

iPerf3 Integration

Reliable integration of iPerf3 functionality in an Android app posed significant challenges with handling network interfaces and maintaining consistent connections.

Solution

Utilized native iPerf3 compilation through JNI with careful error handling and connection state management. Created a robust abstraction layer to handle various network states.

Challenge

Background Services

Ensuring stable operation of network tests and iPerf3 server in background on various Android versions and devices with aggressive power saving.

Solution

Implemented Foreground Services with clear user notifications and comprehensive multi-device testing. Created adaptive power management strategies based on device manufacturer.

Challenge

Data Visualization

Creating informative and smooth graphs/speedometers that update in real-time without compromising UI performance or battery life.

Solution

Optimized UI updates using Coroutines & Flow with efficient rendering cycles. Employed custom Canvas implementations for speedometers and leveraged MPAndroidChart with performance optimizations.
