# 🚀 Earnly Portfolio Page

A modern, animated portfolio website showcasing the Earnly Android application - a comprehensive tax and financial management solution for UK businesses and individuals.

## 📱 About Earnly

**Earnly** transforms complex UK tax calculations into simple, actionable insights. Track your self-assessment, corporation tax, and integrate seamlessly with your favorite accounting software.

### Key Features
- 📊 **Self-Assessment Calculator** - AI-powered tax calculations with HMRC-approved rates
- 🏢 **Corporation Tax Tracking** - Real-time monitoring and forecasting
- 🔗 **HMRC Integration** - Direct connection to official systems
- 💼 **Accounting Software Sync** - QuickBooks, Xero, and Sage integration
- 📈 **Financial Dashboard** - Comprehensive financial overview
- 💳 **Transaction Management** - Smart categorization with OCR

## 🚀 Quick Start

### View Portfolio
1. Open `index.html` in a modern web browser
2. Or serve using a local server:
   ```bash
   python -m http.server 8000
   # Visit http://localhost:8000
   ```

### Deployment Options

#### GitHub Pages
1. Push to GitHub repository
2. Go to Settings → Pages
3. Select source branch
4. Access at `https://yourusername.github.io/earnly-portfolio`

#### Netlify
1. Visit [netlify.com](https://netlify.com)
2. Drag and drop the portfolio folder
3. Get instant deployment

#### Vercel
```bash
npm i -g vercel
vercel
```

## 🎨 Customization

### Brand Colors
Edit in `styles.css`:
```css
:root {
    --primary-blue: #004AAD;
    --secondary-green: #00BF63;
    --secondary-orange: #FF954D;
}
```

### Content
- Update text in `index.html`
- Replace placeholder images
- Add real screenshots
- Update links and CTAs

### Animations
Modify animation timings in `styles.css`:
```css
@keyframes fadeInUp {
    /* Adjust animation parameters */
}
```
