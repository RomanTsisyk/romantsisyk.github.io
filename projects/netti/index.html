Key Features

All the tools you need for professional network testing in one app

⚡

Speed Test

Measure download and upload speeds with detailed statistics. Features a circular speedometer with real-time animations and graphs.

🔧

iPerf3 Client & Server

Professional bandwidth testing tool. Supports TCP/UDP protocols with customizable test parameters and comprehensive results.

📡

Ping Tool

Network latency diagnostics with visual results. Track packet loss, jitter, and round-trip time statistics.

🗺️

Trace Route

Trace packet paths through the network with hop-by-hop visualization and detailed information for each node.

🌐

Network Information

Detailed connection info for Wi-Fi, Cellular, and Ethernet. Shows IP addresses, DNS servers, signal strength, and more.

📊

Data Transfer Rate

Real-time monitoring of data transfer speeds with graphs and historical data tracking.
