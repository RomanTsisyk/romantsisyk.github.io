# mSKOK Portfolio Page

## 🚀 Overview

A modern, animated portfolio landing page for **mSKOK** - a cutting-edge mobile banking application built with Flutter. This portfolio showcases the app's features, technology stack, and provides download links for potential users.

## 🎨 Features

### Visual Design
- **Dark Theme**: Professional dark background with gradient accents
- **Animated Gradient Orbs**: Floating background elements with smooth animations
- **Glass Morphism**: Modern glass effect on cards and UI elements
- **Responsive Design**: Fully responsive across all devices
- **Smooth Animations**: 60fps animations for optimal user experience

### Sections
1. **Hero Section**: Eye-catching introduction with phone mockup
2. **Features Grid**: 6 key features with hover effects
3. **Tech Stack**: Technologies used in development
4. **Screenshots Carousel**: App interface previews
5. **Metrics**: Key performance indicators
6. **Download CTA**: App store download links

## 📱 About mSKOK

mSKOK is a comprehensive mobile banking solution offering:

- **Credit Card Management**: Full control over multiple cards
- **Smart Transfers**: Instant and scheduled payments
- **Savings Goals**: AI-powered savings recommendations
- **QR Code Payments**: Scan-to-pay functionality
- **Biometric Security**: Fingerprint and face recognition
- **Multi-Language Support**: Available in Polish and English

## 🛠️ Technology Stack

### Frontend (Mobile App)
- **Flutter**: Cross-platform framework
- **Dart**: Programming language
- **BLoC Pattern**: State management
- **Retrofit**: API integration

### Backend & Services
- **Firebase**: Analytics, Crashlytics, Core services
- **Azure DevOps**: CI/CD pipeline
- **Local Authentication**: Biometric security
- **Secure Storage**: Encrypted data storage

## 🚀 Deployment

### Quick Deploy Options

#### GitHub Pages
1. Push to GitHub repository
2. Go to Settings > Pages
3. Select source branch
4. Your site will be available at `https://[username].github.io/[repo-name]`

#### Netlify
1. Visit [netlify.com](https://netlify.com)
2. Drag and drop the portfolio folder
3. Get instant deployment with custom URL

#### Vercel
1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` in portfolio directory
3. Follow prompts for deployment

### Local Development
```bash
# Clone the repository
git clone [your-repo-url]

# Navigate to portfolio folder
cd portfolio

# Open in browser
open index.html
# or use a local server
python -m http.server 8000
```

## 🎨 Customization

### Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --primary-color: #4F46E5;
    --secondary-color: #06B6D4;
    --accent-color: #F59E0B;
    --dark-bg: #0A0E27;
}
```
