# 🎯 Portfolio Navigation System - Documentation

## Overview
Єдина навігаційна система для всіх проектів портфоліо, що забезпечує безшовну навігацію між проектами та головною сторінкою.

## ✅ Впроваджені функції (Фаза 2)

### 1. **Глобальний хедер** (`project-nav`)
- ✅ Кнопка повернення на головну
- ✅ Швидкий перемикач проектів (dropdown)
- ✅ Навігація по секціях поточного проекту
- ✅ Адаптивне мобільне меню

### 2. **Bottom Navigation** (`project-bottom-nav`)
- ✅ Перехід до попереднього проекту
- ✅ Кнопка "Всі проекти"
- ✅ Перехід до наступного проекту
- ✅ Адаптивний дизайн для мобільних

### 3. **Клавіатурна навігація**
- ✅ `Ctrl/Cmd + ←` - попередній проект
- ✅ `Ctrl/Cmd + →` - наступний проект
- ✅ `Ctrl/Cmd + H` - повернення на головну
- ✅ `Esc` - закриття меню

### 4. **Мобільні жести**
- ✅ Swipe left - наступний проект
- ✅ Swipe right - попередній проект
- ✅ Візуальні індикатори свайпів

### 5. **Додаткові функції**
- ✅ Progress indicator (позиція в списку проектів)
- ✅ ScrollSpy для активних секцій
- ✅ Smooth scroll до секцій
- ✅ Анімації переходів

## 🚀 Використання

### Для нового проекту:

1. **Скопіюйте шаблон:**
```bash
cp projects/_template.html projects/your-project/index.html
```

2. **Підключіть необхідні стилі:**
```html
<!-- Global Styles -->
<link rel="stylesheet" href="../../assets/css/variables.css">
<link rel="stylesheet" href="../../assets/css/global.css">
<link rel="stylesheet" href="../../assets/css/components.css">
<link rel="stylesheet" href="../../assets/css/project-page.css">

<!-- Your Project Styles (optional) -->
<link rel="stylesheet" href="styles.css">
```

3. **Підключіть скрипти:**
```html
<!-- Global Scripts -->
<script src="../../assets/js/global.js"></script>
<script src="../../assets/js/navigation.js"></script>
```

4. **Додайте проект до списку в navigation.js:**
```javascript
this.projects = [
    // ... existing projects
    { 
        id: 'your-project', 
        name: 'Your Project Name', 
        path: 'your-project', 
        color: '#YourColor' 
    }
];
```

## 🎨 Кастомізація

### Зміна кольорів проекту:
```css
:root {
    --project-primary: #007AFF;
    --project-secondary: #FE6D33;
}
```

### Додавання власних секцій:
```html
<section id="custom-section">
    <!-- Your content -->
</section>
```

Не забудьте додати посилання в навігацію:
```javascript
// В navigation.js, секція createProjectHeader()
<a href="#custom-section" class="section-link">Custom</a>
```

## 📱 Мобільна адаптація

Навігація автоматично адаптується:
- **Desktop (>1024px)**: Повна навігація з усіма елементами
- **Tablet (768-1024px)**: Приховані секційні посилання, активне мобільне меню
- **Mobile (<768px)**: Компактна навігація, bottom bar внизу екрану

## ⚡ API навігації

### Методи класу PortfolioNavigation:

```javascript
// Отримати поточний проект
navigation.currentProject

// Показати індикатор свайпу
navigation.showSwipeIndicator('next', 'Project Name')

// Показати прогрес-бар
navigation.showProgressIndicator()

// Програмна навігація
window.location.href = '../next-project/';
```

## 🐛 Вирішення проблем

### Навігація не з'являється:
1. Перевірте підключення `navigation.js`
2. Переконайтеся, що проект доданий до списку проектів
3. Перевірте консоль на помилки

### Не працюють клавіатурні скорочення:
- На Mac використовуйте `Cmd` замість `Ctrl`
- Переконайтеся, що фокус на вікні браузера

### Проблеми з мобільними жестами:
- Жести працюють тільки на сенсорних пристроях
- Свайп має бути достатньо довгим (>50px)
