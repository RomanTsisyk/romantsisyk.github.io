# Netti - Portfolio Demo Page

## 📱 About the Project

**Netti** is a professional Android application for network testing and monitoring. Built with cutting-edge Android development technologies and Material Design 3.

## 🎨 Portfolio Demo Page

This demo page is created to showcase the Netti app in your developer portfolio. It demonstrates:

### Main Sections:

1. **Hero Section** - Main screen with animated title and phone mockup
2. **Features** - 6 key app features with icons and descriptions
3. **Tech Stack** - Technologies used (Kotlin, Jetpack Compose, Hilt, etc.)
4. **Screenshots Gallery** - Interface screenshot gallery
5. **Design System** - App color palette
6. **Metrics** - Project statistics

## 🚀 Demo Page Features

### Animations and Effects:
- **Animated background** with gradient spheres
- **Floating animation** for phone mockup
- **Shimmer effect** for gradient text
- **Hover effects** for all interactive elements
- **Scroll animations** for feature cards
- **Loading screen** on page load

### Design:
- **Dark theme** matching the app design
- **Gradient accents** for important elements
- **Glassmorphism effects** for cards
- **Responsive design** for all screen sizes

## 🛠️ App Technical Details

### Architecture:
- **Clean Architecture** with layer separation (presentation, domain, data)
- **MVVM** pattern with ViewModel and StateFlow
- **Modular structure** (app, core, domain, data, features)

### Main Components:
```kotlin
- MainActivity.kt - main activity with Splash Screen
- HomeScreen.kt - home screen with service grid
- IperfClientScreenEnhanced.kt - iPerf3 client screen
- SpeedTestScreen.kt - speed testing
- NetworkInfoScreen.kt - network information
```

### Functionality:
1. **Speed Test** - internet speed measurement
2. **iPerf3 Client/Server** - professional network testing
3. **Ping Tool** - latency diagnostics
4. **Trace Route** - packet route tracking
5. **Network Information** - detailed connection info
6. **Data Transfer Rate** - transfer speed monitoring
