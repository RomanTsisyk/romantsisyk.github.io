Installation Guide

Learn how to install the AccessibleMeetings extension on your Chrome browser.

System Requirements

Before installing AccessibleMeetings, make sure your system meets the following requirements:

  • Google Chrome browser (version 89 or later)
  • Working microphone
  • Permissions to access the microphone in your browser

Installation Steps

Follow these simple steps to install AccessibleMeetings:

  1. 1

    Visit the Chrome Web Store

    Go to the AccessibleMeetings page on the Chrome Web Store.

  2. 2

    Add to Chrome

    Click the "Add to Chrome" button at the top right of the page.

  3. 3

    Review Permissions

    A dialog will appear showing the permissions required. Click "Add extension" to continue.

  4. 4

    Confirm Installation

    The AccessibleMeetings icon will appear in your browser toolbar, indicating successful installation.

Installation Screenshot

Example of the Chrome Web Store installation page for AccessibleMeetings

Important Note

AccessibleMeetings requires microphone access to transcribe speech. You'll be prompted to grant this permission the first time you use the extension in a meeting.

After Installation

Once installed, you're ready to use AccessibleMeetings in your online meetings. Here's what to do next:

Verify Installation

Click the AccessibleMeetings icon in your toolbar. A popup should appear showing the extension is ready to use.

Configure Settings (Optional)

You can configure your preferred text size, display mode, and other settings before joining a meeting.

Join a Meeting

Join your Google Meet, Microsoft Teams, or Zoom meeting as you normally would.

Start Transcribing

Click the AccessibleMeetings icon and select "Start Transcribing" to begin live transcription.
