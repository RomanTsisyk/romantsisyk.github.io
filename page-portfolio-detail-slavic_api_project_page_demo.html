SLAVIC API Project

The SLAVIC API Project is a comprehensive collection of Slavic mythology deities, showcasing a unique blend of ancient folklore with modern representation. This API provides detailed information about various Slavic deities, including their domains, symbols, and elements, all formatted in a well-structured JSON format.

Available Repositories

API

This repository contains a detailed JSON structure of Slavic deities. The data includes various attributes like domains, symbols, and elements, providing a rich source for those interested in Slavic mythology.

DemoApp

An Android demo app that showcases Slavic deities through the Slavic Mythology API. It offers a rich exploration experience for users, allowing them to interact with the data effortlessly.

Chrome Extension

Discover the divine with a Chrome extension that presents a vivid catalog of Slavic deities, complete with lore, imagery, and ancient tales.

SlavicAPI-AndroidDemo

This repository, forked from Roman Tsisyk's demo, highlights the features and capabilities of the Slavic API in an Android environment. It’s an excellent tool for developers to understand and integrate the API.

slavic-api.github.io

The project’s main website, providing access to all resources, documentation, and updates related to the SLAVIC API Project.
